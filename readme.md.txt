# Formulario Institucional de Retorno - ANAM

Sistema web para generación automática de correos institucionales
compatible con Outlook Web y Aplicación Outlook.

## Características

- Generación automática de folios únicos
- Bloqueo de múltiples envíos accidentales
- Compatible con computadora y celular
- Apertura directa en Outlook Web o App

## Uso

1. Llenar el formulario
2. Presionar "Generar Correo"
3. Seleccionar método de envío
